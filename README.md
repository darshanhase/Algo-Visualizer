# Algo-Visualizer
Interactive algorithm visualizer for sorting and searching. Includes Bubble, Merge, Quick, Heap Sort, and Linear/Binary Search with animated steps, speed control, and clean UI.

AlgoViz is a fully interactive visualization tool for understanding how **sorting** and **searching algorithms** work internally.  
Built using **HTML, CSS, and JavaScript**, the project visually demonstrates step-by-step execution of classic algorithms with animations, color transitions, and speed controls.
